# portal-uedsports-guide

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面主要用于展示与特定主题相关的信息、工具或指南，不针对任何单一域名或网站。仓库中的页面以静态 HTML 形式存放，便于直接访问、分享或嵌入。

## 目录说明

仓库根目录下包含以下内容：

- `index.html`：仓库的入口页面，通常用于展示目录或导航。
- `pages/`：存放所有独立 HTML 页面的子目录，每个页面为一个独立的 HTML 文件。
- `assets/`：存放页面所需的公共资源文件，如样式表（CSS）、脚本（JavaScript）、图片等。
- `README.md`：本文件，用于说明仓库用途和结构。

## 页面归档说明

所有 HTML 页面均以纯静态方式组织，不依赖后端服务或数据库。每个页面独立运行，可离线查看或部署到任意静态服务器。页面内容遵循通用 Web 标准，兼容主流浏览器。

页面命名建议采用简洁、描述性的英文或拼音，方便直接通过 URL 访问。如需更新或新增页面，请遵循以下原则：

- 保持页面风格一致，避免引入外部依赖过多。
- 在 `pages/` 目录下创建新文件，并在必要时更新 `index.html` 中的链接。
- 如有共用资源，请统一存放在 `assets/` 目录下，避免重复。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意页面。
3. 也可将整个仓库部署到任意静态托管服务（如 GitHub Pages、Netlify 等）进行在线访问。
